# Записи носителей языка

Положите MP3 в подпапки:

- `ce/` — чеченская озвучка (имя файла = slug слова, напр. `marshalla.mp3`)
- `ru/` — русская озвучка

Манифест: `assets/data/audio_manifest.json` — ключ слова → пути `ce` / `ru`.

Запись: 44.1 kHz, mono, без музыки, чистый голос. Для детей — чуть медленнее.

TTS используется автоматически, если файла нет.
